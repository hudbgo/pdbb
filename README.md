# Chill Progress

Una web privada y simple para dos personas. Guarda todo en `localStorage`, así que no necesita backend ni base de datos.

## Qué hace
- Cambiar nombres.
- Crear objetivos por persona.
- Registrar acciones con `+1 hecho`.
- Ver progreso semanal, puntos y rachas.
- Guardar una nota compartida.
- Exportar e importar una copia de seguridad JSON.

## Cómo usarlo
1. Abre `index.html` en el navegador.
2. Rellena nombres y crea objetivos.
3. Sube la carpeta a GitHub y actívalo con GitHub Pages si quieres publicarlo.

## Estructura
- `index.html`
- `styles.css`
- `app.js`

## Nota
Esto es una versión ligera. Los datos se guardan en el navegador de cada dispositivo. Para compartir datos entre dispositivos haría falta una base de datos o sincronización adicional.
